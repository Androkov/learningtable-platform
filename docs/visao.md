
# Visão do Projeto

O Learning Table nasce para resolver um problema recorrente:
a entrega de conteúdo digital de forma fria, confusa e pouco intuitiva.

A proposta é criar uma experiência de leitura fluida, semelhante a um livro,
eliminando PDFs estáticos, áreas de membros complexas e fricção desnecessária.

## Público
- Pessoas 40+
- Leitores não técnicos
- Consumidores de conteúdo digital

## Diretrizes
- A tecnologia não deve aparecer
- O conteúdo é o protagonista
- O acesso deve ser simples
- A experiência deve ser contínua

## Arquitetura
- Learning Table: camada de experiência
- EZ2FLIP: camada de produção
