# Learning Table Platform

Learning Table é uma plataforma de entrega de conteúdo digital focada em
experiência de leitura, eliminando PDFs estáticos e áreas de membros tradicionais.

## Componentes

### Learning Table
Camada visível para o usuário final.
Funciona como biblioteca de leitura, instalável como web app (PWA).

### EZ2FLIP
Engine interna responsável por transformar conteúdos editoriais (PDF)
em experiências de leitura interativas (flipbook).

## Princípios
- Conteúdo acima de ferramenta
- Zero fricção para o usuário
- Ferramentas invisíveis
- Pensado para públicos não técnicos
